## 应用案例

### 双方交易案例
两方（如 a 和 b）之间进行价值的转移。

集群启动后，进入一个 VP 节点。

```sh
$ docker exec -it vp0 bash
```

部署 chaincode example02。

```sh
$ peer chaincode deploy -p github.com/hyperledger/fabric/examples/chaincode/go/chaincode_example02 -c '{"Function":"init", "Args": ["a","100", "b", "200"]}'
```
